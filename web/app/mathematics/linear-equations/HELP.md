# Linear Equations Tool

This interactive tool helps students understand linear equations in the form **y = mx + c**.

## Features

- **Adjust Parameters**: Use the sliders to change the gradient ($m$) and y-intercept ($c$).
- **Visual Aids**: 
  - **Slope Triangle**: Visualizes the rise/run calculation.
  - **Intercepts**: Highlights where the line crosses the x and y axes.
  - **Equation**: Displays the formatted equation on the line.
- **Multiple Lines**: Add up to 5 lines to compare different equations.
- **Presets**: Quickly create parallel, perpendicular, or proportional lines.

## Usage

1. **Changing the Line**: Drag the sliders in the sidebar to modify the equation.
2. **Adding Lines**: Click the "**+ Add**" button in the sidebar to create a new line.
3. **Comparing Lines**: Toggle between lines using the tabs (Line 1, Line 2, etc.) to adjust them individually.
4. **Exporting**: Click "Export" to save your graph as a PNG image or SVG vector file.
